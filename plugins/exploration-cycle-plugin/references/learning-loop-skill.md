# Learning Loop

The learning loop skill lives in the **agent-loops** plugin.

- **Plugin**: `agent-loops`
- **Skill**: `learning-loop`
- **GitHub**: https://github.com/richfrem/agent-plugins-skills/tree/main/plugins/agent-loops/skills/learning-loop
- **When installed**: `.agents/skills/learning-loop/SKILL.md`

The **agent-agentic-os** plugin also provides learning-oriented skills:

- **Skill**: `os-improvement-loop` — retrospective + organic skill/prompt updates
- **Skill**: `os-skill-improvement` — targeted skill refinement
- **GitHub**: https://github.com/richfrem/agent-plugins-skills/tree/main/plugins/agent-agentic-os/skills
- **When installed**: `.agents/skills/os-improvement-loop/SKILL.md`

The **agent-execution-disciplines** plugin provides the execution disciplines
that learning loops refine:

- **GitHub**: https://github.com/richfrem/agent-plugins-skills/tree/main/plugins/agent-execution-disciplines
- **When installed**: `.agents/skills/` (TDD, debugging, verification, code review, git worktrees)

All three plugins install from the same source:
```
github.com/richfrem/agent-plugins-skills
```

Use learning loops when exploration sessions surface recurring friction patterns
that should be codified into improved skills or prompts.
