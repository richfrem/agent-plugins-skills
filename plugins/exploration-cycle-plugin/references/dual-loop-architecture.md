# Dual Loop Architecture

The dual-loop architecture is defined in the **agent-loops** plugin.

All plugins below install from: `github.com/richfrem/agent-plugins-skills`

## agent-loops

- **GitHub**: https://github.com/richfrem/agent-plugins-skills/tree/main/plugins/agent-loops/skills/dual-loop
- **When installed**: `.agents/skills/dual-loop/SKILL.md`

Strategic double-loop: challenges assumptions, reframes the problem, updates
mental models. Tactical single-loop: refines execution within the current
frame. Combined: the agent improves both what it does and how it thinks.

## agent-agentic-os

- **GitHub**: https://github.com/richfrem/agent-plugins-skills/tree/main/plugins/agent-agentic-os/skills/os-improvement-loop
- **When installed**: `.agents/skills/os-improvement-loop/SKILL.md`

The OS improvement loop uses dual-loop principles to evolve agent behavior
across sessions.

## agent-execution-disciplines

- **GitHub**: https://github.com/richfrem/agent-plugins-skills/tree/main/plugins/agent-execution-disciplines
- **When installed**: `.agents/skills/` (TDD, debugging, verification, code review, git worktrees)

Core disciplines that serve as the tactical loop's execution substrate.
