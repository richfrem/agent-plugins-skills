# Dual Loop

The dual-loop skill lives in the **agent-loops** plugin.

- **Plugin**: `agent-loops`
- **Skill**: `dual-loop`
- **GitHub**: https://github.com/richfrem/agent-plugins-skills/tree/main/plugins/agent-loops/skills/dual-loop
- **When installed**: `.agents/skills/dual-loop/SKILL.md`

Dual-loop learning combines strategic (double-loop) and tactical (single-loop)
feedback. Use during exploration when both the approach and the execution
need simultaneous refinement.

All plugins below install from: `github.com/richfrem/agent-plugins-skills`
