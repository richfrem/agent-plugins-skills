# Open Recommendations Tracker

A persistent log of recommendations generated by `synthesize-learnings`. Each entry tracks
the status of a recommendation from discovery through implementation. Append new entries;
never delete or overwrite existing rows.

## Schema

| Date | Title | Target | Priority | Status | Reference |
|------|-------|--------|----------|--------|-----------|
| YYYY-MM-DD | Short recommendation title | scaffolders / specs / analyzer / domain | high / medium / low | open / in-progress / implemented / wont-fix | PR#, commit, or N/A |

**Status values:**
- `open` — logged, not yet picked up
- `in-progress` — being implemented
- `implemented` — done; reference points to the PR or commit
- `wont-fix` — explicitly declined; add a brief reason in the Reference column

## How to Append

When synthesize-learnings generates a new recommendation, append a row here:
```
| [today] | [title] | [target] | [priority] | open | N/A |
```

When the recommendation is implemented, update status and add the reference:
```
| [original date] | [title] | [target] | [priority] | implemented | PR#123 |
```
