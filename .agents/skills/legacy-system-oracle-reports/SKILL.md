# Legacy System Oracle Reports

## Overview

This skill provides workflows for processing Oracle Reports artifacts — extracting queries, parameters, triggers, and database dependencies from `.rdf` XML exports.

## Available Workflows

### Codify
- `/legacy-system-oracle-reports_codify-report`: Generate comprehensive documentation for an Oracle Report

### Investigate
- `/legacy-system-oracle-reports_investigate-report`: Analyze a Report's queries, parameters, and dependencies

## Usage

Invoke these workflows using the slash commands above. They handle context gathering and documentation generation automatically.
