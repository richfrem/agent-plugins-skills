# Acceptance Criteria: skill-improvement-eval

## ✅ Scenarios that must trigger
- "evaluate this skill change"
- "run tests on the new skill"
- "eval the learning loop proposal"
- "check if this modification breaks anything"

## ❌ Scenarios that must NOT trigger
- "what does this skill do?"
- "can you improve my code?"
- "write a new skill for me"
