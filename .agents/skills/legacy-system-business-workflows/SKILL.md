# Business Workflow Management

## Overview

This skill manages the documentation of higher-level Business Workflows (BW-XXXX) that span multiple forms or system components.

## Available Workflows

- `/legacy-system-business-workflows_investigate-business-workflow`: Analyze a process to see if it's already documented.
- `/legacy-system-business-workflows_codify-business-workflow`: Create a new BW-XXXX document describing the end-to-end flow.

## Usage

Use this for multi-step processes (e.g., "Inmate Intake", "Warrant Verification") rather than atomic rules.
