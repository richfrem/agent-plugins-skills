# forms-visualizer

Web-based tool for visually exploring the Oracle Forms dependency graph.

## Overview

The Forms Visualizer is a frontend application that renders the dependency relationships between Forms, Libraries, and Menus as an interactive graph. It is intended for architectural review and impact analysis.

## Usage

There are no CLI workflows to invoke directly. The visualizer is a standalone web application.

Refer to `references/search/implementation_plan.md` for setup and run instructions.
