# Forms Visualizer

## Overview

The Forms Visualizer is a web-based tool for exploring the dependency graph of the Oracle Forms application.

## Documentation

- `docs/setup.md`: Setup instructions.

## Usage

This plugin primarily serves the frontend application code. There are no direct CLI workflows to invoke, but you can consult the documentation here to understand how to run the visualizer locally (usually `npm run dev` in the visualizer directory).
