# screenshot-manager

Utility for organizing and renaming screenshots for documentation.

## Scripts

| Script | Purpose |
|---|---|
| `organize_screenshots.py` | Scan the screenshots directory, rename files per convention, and move them to the correct folder structure |

## Usage

Run after capturing screenshots to ensure they are properly filed before being referenced in documentation.

```bash
python .agent/skills/screenshot-manager/scripts/organize_screenshots.py
```
