# Convert Oracle PLL XML

Converts an exported Oracle Library XML file (`_pll.xml`) into a structured Markdown document.

## Usage
```bash
node ${CLAUDE_PLUGIN_ROOT}/scripts/convert-pll-to-markdown.js --file <input_file> [--out <output_dir>]
```
