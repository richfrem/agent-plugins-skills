# Convert Oracle Forms XML

Converts an exported Oracle Forms XML file (`_fmb.xml`) into a structured Markdown document.

## Usage
```bash
node ${CLAUDE_PLUGIN_ROOT}/scripts/convert-forms-xml-to-markdown.js --file <input_file> [--out <output_dir>]
```
