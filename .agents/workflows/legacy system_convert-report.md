# Convert Oracle Report XML

Converts an exported Oracle Report XML file (`_rdf.xml`) into a structured Markdown document.

## Usage
```bash
node ${CLAUDE_PLUGIN_ROOT}/scripts/convert-report-xml-to-markdown.js --file <input_file> [--out <output_dir>]
```
